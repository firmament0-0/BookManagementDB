Schema

数据库结构

5 张核心表，4 组一对多关系

BookType

TypeID → TypeName, Description

Book

BookID → ISBN, BookName, Author, TypeID, PublisherID, TotalCount, AvailableCount

Publisher

PublisherID → PublisherName, ContactPhone, Address

BorrowRecord

BorrowID → BookID, ReaderID, BorrowDate, DueDate, ReturnDate, Status, Fine

Reader

ReaderID → ReaderName, Gender, Education, Phone, RegisterDate, Status

关系主线： Reader 通过 BorrowRecord 借阅 Book；Book 通过 TypeID 与 PublisherID 关联图书类型和出版社。
