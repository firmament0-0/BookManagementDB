# 图书借阅管理系统前端演示界面

这个目录是为数据库课程设计补充的前端演示界面，对应数据库 `BookBorrowDB` 的 5 张核心表：`BookType`、`Publisher`、`Book`、`Reader`、`BorrowRecord`。

## 使用技术

- HTML5：页面结构和表格、表单、模块布局。
- CSS3：蓝白主色视觉、响应式布局、状态标签、提醒色和统计条形图。
- 原生 JavaScript：模块切换、搜索筛选、借书登记、还书登记、统计刷新和本地演示数据保存。
- LocalStorage：把演示过程中的借书、还书变化临时保存在浏览器中，刷新页面不会马上丢失。

## 打开方式

直接用浏览器打开 `frontend/index.html` 即可，不需要安装依赖，也不需要启动后端服务。

页面内部不展示技术说明，也不引用 SQL 运行截图，方便直接用于系统演示。

## 已实现的演示功能

- 总览：馆藏总量、当前可借、未还记录、逾期提醒。
- 图书管理：按关键字和图书类型筛选图书，支持新增图书和库存更新。
- 读者管理：展示读者状态、联系方式、借阅次数和当前未还数量，支持新增读者和状态更新。
- 借阅管理：模拟借书登记和还书登记，并同步更新可借数量与罚款金额。
- 查询统计：展示类型馆藏数量、热门图书排行、读者借阅次数和最近数据变化。
- 数据库结构：用界面化方式说明 5 张表的关系和主线。
